# はじめに

本ドキュメントは、音楽ゲームosu!のmaniaモード、key=7で使用するコレクション管理ツール「**NakuruTool**」の仕様を定義するものです。初心者でも使いやすくモダンなUIでコレクションの管理機能を提供することを目的にツールの開発をおこないます。

```
NakuruTool
```

## 1. プロジェクト概要

  - **ツール名**: NakuruTool
  - **目的**: コレクション管理を行うツール

## 2. 主な機能概要
1. jsonファイルへのコレクションのエクスポート機能
2. jsonファイルへのコレクションのインポート機能
3. コレクション内容のリストでの閲覧機能
